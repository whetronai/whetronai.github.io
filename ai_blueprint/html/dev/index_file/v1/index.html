計畫摘要

本提案旨在為輝創電子股份有限公司（以下簡稱「輝創電子」）擘劃一套全面的智慧轉型藍圖，透過導入業界最先進的人工智慧（AI）技術（包含生成式 AI、深度學習、機器學習）、以 NVIDIA Omniverse 平台為基礎的數位雙生（Digital Twin）技術，以及 AI 協作型機器人（Cobots），全面革新產品設計研發、生產製造工廠場域以及涵蓋產、銷、人、發、財、資、內稽內控九大循環的行政營運管理。

輝創電子目前在倒車雷達系統、ISG、ADAS、AVM、APA、BSD 等車用電子領域已奠定良好基礎，並擁有 ERP (SAP)、PLM、MES 等成熟資訊系統。本計畫將在此基礎上，透過前瞻技術的整合應用，目標實現公司整體效率與效能的顯著提升，推動產品、服務、製程及行政管理邁向高度智慧化規模。

預期效益包括大幅降低營運成本、提升生產效率、增強產品創新能力，最終達到提升企業整體獲利能力與市場競爭力的戰略目標。本計畫預計分三階段於三至五年內完成，總體預估經費約為新台幣三千萬元。

核心技術應用場景

AI輔助設計與模擬

導入生成式AI工具，協助工程師快速生成多種ADAS產品的初步設計概念，並進行設計迭代。利用機器學習分析過往產品數據與市場回饋，優化新產品的規格與功能參數。

效益：縮短設計週期15-25%

產品數位雙生

針對ADAS系統、AVM環景系統，在Omniverse中建立高擬真數位雙生模型。模擬產品在各種真實世界條件下的運作表現，如不同光照、天氣對攝影機與雷達性能的影響。

效益：降低實體樣品與測試成本20-30%

智慧研發知識庫

利用自然語言處理(NLP)及機器學習技術，整合公司內部研發文檔、專利資料、測試報告、市場資訊等，建立智慧研發知識庫，供研發人員快速查詢、獲取洞見。

效益：加速研發決策流程，提高創新效率

感測器模擬與數據生成

在Omniverse建立的虛擬場景中模擬光達、雷達、攝影機等感測器的行為，生成大量用於訓練ADAS演算法的合成數據，彌補真實世界數據採集的不足。

效益：加速AI模型訓練，提升演算法可靠性

工廠數位雙生建置

以輝創電子的生產廠房為藍本，運用Omniverse逐步建構整個工廠或關鍵生產線的數位雙生模型。整合來自MES、PLC、SCADA及IoT感測器的即時數據。

效益：提升生產效率與產能利用率約10-20%

AI預測性維護

在生產線上關鍵設備（如SMT產線設備、自動化組裝機）部署感測器，收集運作數據，透過機器學習模型預測設備潛在故障，實現預測性維護，減少非預期停機。

效益：減少設備非預期停機時間達20-30%

AI視覺檢測

導入基於深度學習的AI視覺檢測系統，自動辨識產品外觀瑕疵、組裝缺陷（如電子元件焊接點、機構件密合度），提升檢測效率與準確度，並與現有MES系統整合。

效益：降低產品不良率約15-20%

AI協作型機器人導入

在輝創電子的車用電子產品組裝線，導入AI協作型機器人執行精密鎖付、插件、塗膠、標籤黏貼等任務，尤其適用於少量多樣或客製化產品線。

效益：提高組裝與檢測的質量一致性與生產效率約10-20%

不同部署方式比較

本地重點部署

優勢：數據安全與系統自主性有較高掌控，長期TCO可能較具優勢

劣勢：初期硬體投資較高，擴展性受限

總經費: 約3,000萬元

混合雲部署 (建議方案)

優勢：兼顧安全性、效能、彈性與成本效益，初期資本支出較低

劣勢：架構複雜度較高，需管理混合環境

總經費: 約2,950萬元

雲端重點部署

優勢：高度擴展彈性，可快速取用最新AI雲端服務

劣勢：長期雲端服務訂閱費用較高，數據傳輸成本高

總經費: 約3,050萬元

OGSM 策略框架

總體目標 (Objective)

透過導入尖端AI技術、數位雙生與AI協作型機器人，全面提升輝創電子在產品研發、生產製造及行政營運的智慧化水準，顯著強化企業核心競爭力，實現永續獲利成長，成為全球汽車電子產業智慧化解決方案的領導者。

具體目標 (Goals)

G1 研發創新領先

未來三年內，新產品（特別是ADAS、AVM、APA相關）平均研發週期縮短20%，因設計優化帶來的直接成本降低10%，專利申請數量提升15%。

G2 智慧製造卓越

未來三年內，主要生產廠區整體設備效率(OEE)提升15%，產品首次通過率(FPY)提升5%，單位產品製造成本降低8%。

G3 營運效能躍升

未來三年內，關鍵行政作業平均處理時間縮短30%，行政營運費用佔營收比重降低1個百分點。

G4 企業價值增長

未來五年內，本專案導入後，公司整體淨利率提升2-3%，客戶滿意度提升10%，成功拓展至少一個新興智慧化產品市場。

策略 (Strategies)

S1 技術賦能平台化

  • • 建立以NVIDIA Omniverse為核心的企業級數位雙生協作平台
  • • 全面導入並深化AI技術應用
  • • 階段性部署AI協作型機器人

S2 流程再造智慧化

  • • 以數位雙生驅動產品研發流程變革
  • • 打造數據驅動的智慧工廠運營模式
  • • 透過AI與RPA技術，全面檢視行政流程

S3 人才培育與文化轉型

  • • 成立跨部門「智慧轉型推動辦公室」
  • • 實施全面的數位技能提升計畫
  • • 塑造擁抱變革、鼓勵創新的企業文化

總結與行動建議

透過實施本智慧轉型藍圖，輝創電子將能夠顯著提升研發創新能力、生產製造效率與行政運作效能，進而強化企業整體競爭力。基於上述分析，我們建議：

  1. 以試點專案快速驗證方式開始轉型，累積成功經驗與信心
  2. 採用混合雲架構，兼顧成本效益性與系統靈活性
  3. 優先導入可快速見效的AI預測性維護與視覺檢測系統
  4. 持續投資於數位雙生技術，為未來的創新奠定基礎
  5. 重視員工數位轉型培訓，使技術與人才同步發展

面對日新月異的競爭環境，及早啟動智慧轉型已是當務之急。透過系統性、階段性的導入計畫，輝創電子不僅能降本增效，更能開創全新商業模式與市場機會，邁向更繁榮的未來。
